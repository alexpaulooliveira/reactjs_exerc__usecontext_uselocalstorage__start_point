# ReactJS - Hooks - useContext

## Exercício

Execute as seguintes tarefas:

### 1) Crie um software que guarde as seguintes variáveis utilizando o Hook useContext:

lang <br/>
theme <br/>
auth <br/>

Essas variáveis devem ter, inicialmente, os seguintes valores:

lang = 'en' <br/>
theme = 'dark' <br/>
auth = 'off' <br/>
